# Final Report Template

Use this structure for the innovation note, dataset note, and final narration.

## 1. Title

Short project title that mentions brain tumour detection and GAN augmentation.

## 2. Problem statement

Explain the data scarcity and clinical motivation.

## 3. Proposed approach

Describe the detection, segmentation, classification, and GAN augmentation pipeline.

## 4. Dataset summary

Describe sources, modalities, image counts, mask counts, labels, and preprocessing.

## 5. Innovation

Highlight the novelty, such as conditional generation, attention, or augmentation strategy.

## 6. Evaluation

Report baseline versus augmented metrics, FID, FS/F1, and segmentation quality.

## 7. Deployment

Describe the Streamlit app and how the model will be demonstrated.

## 8. Conclusion

Summarize the value of the project and any limitations.
