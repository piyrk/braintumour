# Phase 3: Baseline Models

## Goal

Establish the reference performance for all supervised tracks.

## Models

- Detection CNN
- U-Net segmentation model
- Multi-class tumour classifier

## Outputs

- Model checkpoints
- Training logs
- Confusion matrix for detection/classification
- Dice and IoU for segmentation
- Baseline metric table

## Comparison rule

All later work should be compared against these baselines, especially the GAN augmentation track.
