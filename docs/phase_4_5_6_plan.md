# Phases 4 to 6: GAN, Augmentation, and Reporting

## Phase 4: GAN development

- Build generator and discriminator.
- Train on unlabeled MRI data.
- Save sample grids, history, FID curve, and checkpoints.

## Phase 5: GAN augmentation

- Generate synthetic MRI samples.
- Retrain the classifier with augmented data.
- Compare against the baseline classifier.

## Phase 6: Evaluation and reporting

- Report detection, segmentation, and classification metrics.
- Report FID and FS/F1.
- Prepare the dataset note and innovation note.

## Output expectation

- A single comparison table showing baseline versus augmented performance.
- A final metrics summary for the submission package.
